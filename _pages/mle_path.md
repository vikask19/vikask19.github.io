A running reference for engineers preparing for MLE interviews. MLE loops typically test coding, ML fundamentals, ML system design, project depth, and behavioral signals — in varying weight depending on the company.

---

## The Interview Loop

Most MLE loops contain some combination of:

- **Algorithmic coding:** LeetCode-style data structures and algorithms.
- **ML fundamentals:** classical ML, deep learning basics, metrics, optimization, and model debugging.
- **ML coding:** implement common pieces from scratch — logistic regression, self-attention, batch normalization, evaluation metrics.
- **ML system design:** design an end-to-end production ML system covering data, modeling, serving, monitoring, and iteration.
- **Project deep dive:** explain one or two projects deeply, including failures, tradeoffs, metrics, and business impact.
- **Behavioral:** crisp stories about leadership, conflict, ambiguity, execution, and learning.
